Description détaillée

RCC PLL configuration structure definition.

Documentation des champs

◆ PLLM

uint32_t PLLM

PLLM: Division factor for PLL VCO input clock. This parameter must be a value of PLLM Clock Divider

◆ PLLN

uint32_t PLLN

PLLN: Multiplication factor for PLL VCO output clock. This parameter must be a number between Min_Data = 8 and Max_Data = 127

◆ PLLP

uint32_t PLLP

PLLP: Division factor for ADC clock. This parameter must be a value of PLLP Clock Divider

◆ PLLQ

uint32_t PLLQ

PLLQ: Division factor for SAI, I2S, USB, FDCAN and QUADSPI clocks. This parameter must be a value of PLLQ Clock Divider

◆ PLLR

uint32_t PLLR

PLLR: Division for the main system clock. User have to set the PLLR parameter correctly to not exceed max frequency 170MHZ. This parameter must be a value of PLLR Clock Divider

◆ PLLSource

uint32_t PLLSource

RCC_PLLSource: PLL entry clock source. This parameter must be a value of PLL Clock Source

◆ PLLState

uint32_t PLLState

The new state of the PLL. This parameter can be a value of PLL Config
