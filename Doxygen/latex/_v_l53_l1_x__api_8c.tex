This function starts the ranging distance operation~\newline
The ranging operation is continuous. The clear interrupt has to be done after each get data to allow the interrupt to raise when the next data is ready~\newline
1=active high (default), 0=active low, use Set\+Interrupt\+Polarity() to change the interrupt polarity if required.
